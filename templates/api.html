API Reference

Operational endpoints exposed by MiniWeb for monitoring and documentation services.

GET/api/metrics

Returns consolidated system metrics: load averages, memory, disks, network interfaces and process counters.

GET/api/man/sections

Lists available manual areas and sections currently indexed by MiniWeb.

GET/api/man/{area}/{section}

Returns all pages in a section. Useful for dynamic section explorers.

GET/api/man/search/{query}

Performs full-text search against manual pages and returns apropos-like results.

GET/api/networking

Networking info JSON API, retireve: routes, DNS, interface statistics.

GET/api/packages/search?q={query}

Search installed packages by package name pattern.

GET/api/packages/info?name={package}

Returns detailed package information for a single package.

GET/api/packages/which?path={absolute_file_path}

Returns package(s) that own the provided absolute file path.

GET/static/benchmark.html

Benchmarks

Last benchmark.sh execution results.
